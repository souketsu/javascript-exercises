# 练习08 - removeFromArray

实现一个函数，该函数接收一个数组和其他一些参数，然后从该数组中移除这些参数：

```javascript
removeFromArray([1, 2, 3, 4], 3); // 应该移除3并返回[1,2,4]
```

看看你是否能在这个练习中使用一些内置的数组方法。

## 提示

这个练习的第一个测试相当简单，但对于后面的测试，有几个需要思考（或搜索）的问题：

- 你可以操作传入函数的原始数组，或者创建一个新数组作为结果返回。
- 如何从数组中移除单个元素
- 如何处理JavaScript函数中的多个可选参数
- 更多信息，请查看MDN关于[函数参数](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Functions/arguments)的页面 - 向下滚动到关于`Array.from`或展开运算符的部分。你也可以查看MDN关于[剩余参数](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Functions/rest_parameters)的页面。
