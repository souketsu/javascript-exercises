# 练习 04 - mathEquations

将 `=` 右侧的字符串替换为它们所描述的表达式。**不要**手动输入方程的答案。

目前，变量都是字符串。它们需要变成数学方程。

为了帮助你，第一个变量已经为你完成：

```js
const a = 1 - 1 // 免费赠品！！！这是"一减一"的答案
```

以下是一个**错误**答案的例子：

```js
const a = 0 // 错误！！！你需要写出方程，而不是直接放入方程的结果！
```

现在就可以运行测试套件，看看第一个测试是否通过，然后从那里继续。祝你好运！
